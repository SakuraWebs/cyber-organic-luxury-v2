## 💎 El Proyecto

**Cyber-Organic Luxury** es una experiencia digital premium diseñada para una agencia boutique de vanguardia. El sitio fusiona la precisión de la inteligencia artificial con la fluidez de la vida biológica, creando un entorno visualmente impactante y altamente interactivo.

### ✨ Características Principales
- **Estética Ciber-Orgánica:** Paleta de colores profunda con acentos bioluminiscentes (Cian Eléctrico y Bronce Oxidado).
- **Navegación Fluida:** Smooth scrolling y transiciones de página dinámicas con `framer-motion`.
- **Arquitectura Moderna:** Desarrollado con **React 19** y **Vite 6** para un rendimiento ultrarrápido.
- **Componentes de Lujo:** Preloader cinemático, chatbot inteligente integrado y portafolio dinámico.

---

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React 19, TypeScript
- **Estilos:** Tailwind CSS 4 (Custom Theme)
- **Animaciones:** Framer Motion (motion/react)
- **Iconografía:** Lucide React
- **Ruteo:** React Router Dom
